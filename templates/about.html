Project Overview: Rising Waters

An end-to-end Machine Learning web application designed to predict flood probabilities using meteorological and meteorological observations.

Academic & Internship Objective

This project represents a complete, production-ready machine learning workspace suitable for an internship final review. It spans the entire data science cycle: ingesting records, preprocessing and cleaning data (handling nulls, dropping duplicates, removing outliers via the Interquartile Range method), scaling features, training 4 classifiers, evaluating cross-metrics (F1, Accuracy, Precision, Recall, ROC-AUC), saving model weight bins, and exposing them through a Flask application.

Weather Feature Schema

Feature Column Description Units / Scale
Annual_Rainfall Cumulative regional rainfall over 12 months Millimeters (mm)
Monsoon_Intensity Index of wet season monsoonal wind and storm strength Scale (0 - 10)
Cloud_Coverage Atmospheric cloud saturation percentage Percent (0% - 100%)
Humidity Atmospheric relative moisture saturation Percent (0% - 100%)
Temperature Average ambient air temperature Degrees Celsius (°C)
River_Discharge Volume rate of river water flow flowing past a node m³/second
Flood Target classification variable (1 = Flood, 0 = Safe) Binary Class
Technology Stack
  • Backend Core: Python 3.10+ & Flask
  • Data processing: Pandas & NumPy
  • ML Modeling: Scikit-Learn & XGBoost
  • Plots & Heatmaps: Matplotlib & Seaborn
  • Serialization: Joblib (model & scaler bin)
  • Frontend View: HTML5, CSS3, Bootstrap 5, JS
  • Server & Cloud: Gunicorn (IBM Cloud ready)
