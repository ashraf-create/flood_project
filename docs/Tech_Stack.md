# Tech Stack

The application leverages standard Python libraries for its machine learning pipeline and a light modern front-end design:

| Component | Technology | Description |
| :--- | :--- | :--- |
| **Backend Framework** | **Flask** (v2.2+) | Handles server routes, form submissions, page templates, and database logs. |
| **Machine Learning** | **XGBoost** (v1.5+)<br>**Scikit-Learn** (v1.0+) | Classification algorithms, training splits, metric evaluations, and preprocessing. |
| **Data Processing** | **Pandas** (v1.3+)<br>**Numpy** (v1.21+) | Dataframes, matrix manipulations, and array loading. |
| **Visualizations** | **Matplotlib** (v3.5+)<br>**Seaborn** (v0.11+) | Correlation heatmaps, target distributions, ROC curves, and confusion matrices. |
| **Model Serialization** | **Joblib** (v1.1+) | High-performance pickle serializer to save/load weights. |
| **Frontend Styling** | **Bootstrap 5** / **CSS** | Responsive grids, glassmorphism UI components, and premium font typography. |
| **Client Scripting** | **jsPDF** (v2.5+) | Compiles client-side PDF document summaries. |
