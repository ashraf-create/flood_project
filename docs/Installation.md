# Installation

Follow these steps to set up the application environment on your local machine:

### 1. Prerequisites
- **Python**: Version 3.9, 3.10, or 3.11 is required.
- **Git** (Optional): For tracking source code.

### 2. Set Up a Virtual Environment
Create an isolated environment to prevent dependency conflicts:
```bash
python -m venv venv
```
Activate it:
- **Windows (PowerShell)**:
  ```bash
  .\venv\Scripts\activate
  ```
- **macOS/Linux**:
  ```bash
  source venv/bin/activate
  ```

### 3. Install Dependencies
Install all required libraries listed in [requirements.txt](file:///c:/Users/satya/Downloads/flood_project/requirements.txt):
```bash
pip install -r requirements.txt
```
